# HomeWorth (Local Dev)

This is a lightweight Vite + React scaffold containing your HomeWorth UI component in `src/App.jsx`.

Quick start:

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

Open the dev URL shown by Vite (usually http://localhost:5173).

Files created:

- [package.json](package.json)
- [index.html](index.html)
- [src/main.jsx](src/main.jsx)
- [src/App.jsx](src/App.jsx)
